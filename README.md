# CryptoPulse – Real-Time Cryptocurrency Market Tracking Platform


## Project Description
CryptoPulse is a cryptocurrency tracking web application that allows users to monitor cryptocurrency prices, manage portfolios, create watchlists, and track market trends.

## Features
- User Registration and Login
- Cryptocurrency Dashboard
- Portfolio Management
- Watchlist Management
- Market Price Tracking
- User Profile Settings
- Secure Authentication

## Technologies Used
- Java
- JSP
- Servlets
- JDBC
- MySQL
- HTML
- CSS
- JavaScript
- Apache Tomcat

## Project Structure
- Frontend: HTML, CSS, JavaScript, JSP
- Backend: Java Servlets
- Database: MySQL
- Connectivity: JDBC

## Database
MySQL database is used for storing:
- User Information
- Portfolio Data
- Watchlists
- Cryptocurrency Data

## How to Run
1. Import the project into Eclipse.
2. Configure Apache Tomcat.
3. Create the MySQL database.
4. Update database credentials.
5. Run the project on Tomcat Server.
